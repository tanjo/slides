## スープは「豚清湯（ぶたちんたん）」

鶏ガラではなく、豚肉の旨味を極限まで抽出した透明なスープ

表面を厚めのラードが覆い最後まで激熱

キリッとした醤油の塩味が特徴

ツルツル・モチモチとした多加水の平打ち麺。喉越しが抜群

日本一煮干を追求した“ニボい！”ラーメンの「ラーメン凪」が麺を提供している

## 高菜とキムチが食べ放題

虎ノ門ヒルズ ビジネスタワー地下にある「**虎ノ門 たきちゃんラーメン**」

営業時間：10時～23時(土日は21時)

この店舗でご飯を頼まないのは損です。

## 朝から食べられる

新橋駅前ビル1号館地下にある「**新橋ニューともちん**」

営業時間：8時～17時（土日定休日）

なんと朝8時から営業しているため、朝ラーメンが食べられる。
